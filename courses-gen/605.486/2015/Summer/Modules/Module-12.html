Module 12 - NFC

Overview

Some applications are completely self-contained. They contain a user interface that does not require data from outside the device. Other applications need to fetch and/or update data from other devices or the internet.

In this module, we'll reach out using TCP/IP over a wireless connection. We'll communicate using a custom protocol to transfer information and requests between our device and a server on the internet.

Objectives

By the end of this module, you will be able to:

  • Create a Java-based server component that runs on a desktop/laptop computer.
  • Create an Android client component (in an Activity) that connects to the server component.
  • Pass data and commands between the client and server using a custom protocol.

Learning Guide

Begin this module by watching all of the recorded lectures below. I strongly recommend that you watch them in the order listed, as some lectures assume knowledge gained from earlier lectures.
