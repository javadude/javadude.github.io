605.486 - Mobile Application Development for the Android Platform - Fall 2015

Module 30 - Further Resources

Overview

You've made it to the end! But you're still hungering for more!

In this module, we'll explore some useful resources that can help you learn more about Android application programming and find the answers to questions that will come up as you continue to develop applications.

Objectives

By the end of this module, you will be able to:

  • Find new information on Android application development.
  • Follow useful and important forums/blogs containing new developments.
  • Find answers to questions on Android user groups and StackOverflow.

Learning Guide

Begin this module by watching all of the recorded lectures below. I strongly recommend that you watch them in the order listed, as some lectures assume knowledge gained from earlier lectures.

Refer to the Course Outline for any assignment start or due dates!

Please post any questions to the Q & A section of the Discussion Board.
