Assignment A5 - Widgets and Notifications

NOTE: Be sure to name this project and its packages starting with lastname.firstname.hw5.

In this assignment, we'll add notifications and a widget to my to-do application.

Download the starter code from http://javadude.com/courses-static/605.486/2016/Summer/TodoWidgets-Starter.zip

You will need to add support for notifications and a home-screen widget.

For an idea of what the app should look like and how it should behave, please see the class video.

Whenever an item is edited “due”, create, update or cancel a notification

  • (NOTE: Ideally this would be triggered when am item is coming due, but I don't want you to have to mess with your time settings)
  • The notification should be persistent (ongoing)
  • There should be only one notification for the application
  • If there are no due items, cancel the notification
  • The notification should show number of due items
  • The notification should show the names of all due items
  • The notification's main “tap” should open the to-do list activity
  • Include a “new item” button in the notification that jumps directly to the edit activity (be careful of your back stack here; be sure to test when the to-do app isn't active)

Create a homescreen widget that

  • Shows either “No Items Due” or “N Items Due” (where N is the number of items due)
  • When tapped goes to list of all items
  • Has a button for adding a new item (again be careful of your back stack)

Submitting Your Assignment

To submit your assignment:

  1. Right-click your project(s) and choose Export…

  2. Under 'General' choose 'Archive File' and press Next

  3. Make sure the following options are checked:

    • Save in zip format
    • Compress the contents of the file
    • Create directory structure for files

  4. Choose the zip file to create in the To Archive File box.

    *The name of the zip file should be the same as your project name, followed by .zip

  5. Press Finish.

  6. Go to the Blackboard course website

  7. Click on the Assignments link on the left navigation bar

  8. Click on the Assignment A5 link

  9. Attach your submission and enter any notes

  10. Be sure to submit your assignment, not just save it!
