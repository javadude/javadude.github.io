Module 18 - Preferences

605.486 - Mobile Application Development for the Android Platform

Overview

Often you'll want to create settings for an application that can persist across invocations. The Android platform makes it easy to define these settings and a user interface to allow their customization.

In this module, we'll create an XML-based settings definition and a PreferenceFragment to display and edit the settings.

Objectives

By the end of this module, you will be able to:

  • Define settings for an application using an XML file.
  • Create a PreferenceFragment to display and edit those settings.
  • Read and write settings from an Activity.
  • Update the summary descriptions in the settings UI to include set values.
