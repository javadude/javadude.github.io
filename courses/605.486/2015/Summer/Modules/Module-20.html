Module 20 - Property Animation

605.486 - Mobile Application Development for the Android Platform

Overview

Sometimes it's useful to vary a property of an object over time. For example, you may want to move a TextView across the screen.

In this module, we'll use Android's Property Animation to change the values of visual and non-visual objects.

Objectives

By the end of this module, you will be able to:

  • Animate the values of a non-visual object.
  • Use the Observer Pattern to link a non-visual object with an on-screen representation.
  • Animate View locations.
  • Combine animations to create slide-in/out views.

Learning Guide

Begin this module by watching all of the recorded lectures below. I strongly recommend that you watch them in the order listed, as some lectures assume knowledge gained from earlier lectures.
