Module 4 - Lists

605.486 - Mobile Application Development for the Android Platform

Overview

Some user interfaces require a more dynamic approach. Rather than fixed fields in the interface, a scrollable list of data may be preferable.

In this module, we'll discuss the Model-View-Controller pattern and explore Android's list view, creating its data and adding it to the user interface.

Objectives

By the end of this module, you will be able to:

  • Describe Model-View-Controller communication.
  • Describe how Android uses MVC between its ListView and ListAdapter.
  • Create a ListAdapter that provides in-memory data to a list (vs from a database).
  • Create a ListView that displays the data from the ListAdapter.
  • Launch and Activity when items in the ListView are clicked.
