Module 19 - Widgets and Broadcast Receivers

605.486 - Mobile Application Development for the Android Platform

Overview

One of Android's defining features is the ability to have live Widgets on your device's home screens.

In this module, we'll create a simple counting service to send a broadcast Intent with the current count. We'll introduce Broadcast Receiver components that can capture the broadcast Intent and update a home screen widget to display the current count.

Objectives

By the end of this module, you will be able to:

  • Create a Broadcast Receiver.
  • Define a Widget using XML and a WidgetProvider.
  • Update the Widget from the WidgetProvider.
  • Respond to tap actions on the Widget.
