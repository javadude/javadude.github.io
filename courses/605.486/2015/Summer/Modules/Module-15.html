Module 15 - Maps

Overview

The Google Maps API allows your application to display maps, add markers and draw custom graphics based on map locations.

In this module, we'll use the Google Maps API to create a Car Finder application. We'll display a map, allow the user to place a marker representing their car's location, and draw a route from their current location to the marker.

Objectives

By the end of this module, you will be able to:

  • Register online for use of the Google Maps API.
  • Add MapFragments to your application.
  • Display the current location (or any other location).
  • Add markers to the map.
  • Draw custom shapes on the map.
  • Retrieve navigation directions from the Google Maps server.
  • Invoke the Google Navigation application for turn-by-turn directions.
