Module 28 - Speech

Overview

Most applications interact with the user via text and touch. Sometimes the user cannot read the screen (visual disability, driving, hands otherwise occupied, limited screen device such as Google Glass, etc) or interact via touch. Android supports both text-to-speech and speech-to-text interaction.

In this module, we'll create a simple voice-controlled text adventure game to demonstrate both directions of speech/text conversion.

Objectives

By the end of this module, you will be able to:

  • Capture user voice input and recognize it as phrases.
  • Parse the input to determine its meaning.
  • Perform actions based on that input.
  • Report status to the user using text-to-speech synthesis.
  • Set up a modal application that alternates between speaking and listening.
