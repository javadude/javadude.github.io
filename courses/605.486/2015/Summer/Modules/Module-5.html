Module 5 - Action Bars

Overview

User input consists of viewing and editing existing data and triggering actions to perform in an application. While buttons can act as triggers, they don't provide a consistent and intuitive interface across different applications.

In this module, we'll explore Android's ActionBar interface. Similar to toolbars on desktop UIs, the ActionBar is a consistent UI element holding commonly-used actions.

Objectives

By the end of this module, you will be able to:

  • Add an ActionBar to an Activity.
  • Specify actions to be displayed via XML.
  • Create icons representing each action.
  • Respond to invoked actions in your activity.
