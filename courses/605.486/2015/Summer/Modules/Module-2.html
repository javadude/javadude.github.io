Module 2 - Activities

Overview

The primary unit of user interaction in Android is called an “Activity”.

In this module, we'll explore the Activity lifecycle and creation of simple user interface components using simple layouts.

Objectives

By the end of this module, you will be able to:

  • Describe the Android Activity lifecycle.
  • Create a user interface using an Activity with text entry fields and buttons.
  • Design simple user interfaces using Android linear and relative layouts.

Learning Guide

Begin this module by watching all of the recorded lectures below. I strongly recommend that you watch them in the order listed, as some lectures assume knowledge gained from earlier lectures.
