Module 7 - Databases and Content Providers

605.486 - Mobile Application Development for the Android Platform

Overview

We've seen how to store data to deal with screen rotation, but that data does not persist across separate application invocations. Android supports relational database storage using SQLite.

In this module, we'll explore database creation and handling Create/Read/Update/Delete operations to manage your data. To make data access more consistent, the Android platform defines Content Provider components, wrapping data access in a REST-like interface.

Objectives

By the end of this module, you will be able to:

  • Create a database.
  • Manage data in a database via a Content Provider.
  • Link Content Provider results with a ListAdapter for display to the user.
