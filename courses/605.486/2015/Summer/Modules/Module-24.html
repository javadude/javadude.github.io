Module 24 - HTML Applications

605.486 - Mobile Application Development for the Android Platform

Overview

So far, we've focused on native Android applications. An alternative approach is an HTML-based application, which can allow device-independent applications.

In this module, we'll discuss thin client (runs on a server and viewed through a browser) and fat client (runs locally) applications using HTML.

Objectives

By the end of this module, you will be able to:

  • Display web applications (thin and fat client) in a WebView.
  • Manage the user's interaction with the WebView.
  • Communicate between an Activity and HTML pages using JavaScript.
  • Create HTML5 local databases and read/write using JavaScript.

Learning Guide
