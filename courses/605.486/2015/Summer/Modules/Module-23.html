Module 23 - Google Cloud Messaging

605.486 - Mobile Application Development for the Android Platform

Overview

Normally, when an application needs to check for information such as the current weather, it occasionally polls a server to get the latest information. If the information changes frequently, polling is effective. However, if the data infrequently changes, polling wastes data transmission to repeatedly retrieve the same values.

Google Cloud Messaging (GCM) is a push service. Your server communicates with the GCM server to send updates to the devices, reducing data usage.

Objectives

By the end of this module, you will be able to:

  • Describe the data flow for Google Cloud Messaging.
  • Register with GCM  for data services.
  • Create a server application that pushes notifications to GCM, which pushes the notifications to devices.
  • Create Broadcast Receiver components that accept the notifications on the device.
