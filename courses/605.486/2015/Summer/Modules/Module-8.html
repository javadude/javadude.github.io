Module 8 - Fragments

605.486 - Mobile Application Development for the Android Platform

Overview

The Android platform supports many different types of devices and form-factors. A user interface defined on one device may not effectively use the available space on other devices. Fragments separate your user interface into chunks that can be displayed independently, or combined into a single interface depending on device configurations.

In this module, we'll see how to create applications that can adjust to different device configurations to take advantage of additional screen space and reduce the navigation required of the user.

Objectives

By the end of this module, you will be able to:

  • Create Fragment layouts and management classes.
  • Transfer data between Activities and Fragments.
  • Trigger actions in Activities from Fragments.
