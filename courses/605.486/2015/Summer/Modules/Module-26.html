Module 26 - Barcode and QR Code Reading

605.486 - Mobile Application Development for the Android Platform

Overview

Data entry can be tedious and error-prone. One of the ways we've sped that entry and ensured correctness is through the use of various types of graphical codes, such as barcodes and QR codes. With the prevalence of cameras on mobile devices, these codes can be easily read.

In this module, we'll discuss various uses for graphical codes and how they can be read using existing third-party software.

Objectives

By the end of this module, you will be able to:

  • Describe several uses for graphical codes.
  • Install the Zebra Crossing (ZXing) barcode reader application.
  • Create an Activity that invokes the ZXing reader Activity via an Intent to capture and return data from a graphical code.
