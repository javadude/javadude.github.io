Module 1 - Development Environment

Overview

Android development requires installation and use of several tools:

  • The Java Development Kit (JDK)
  • The Android Software Development Kit (SDK)
  • Eclipse Integrated Development Environment (IDE)
  • The Android Developer Tools for Eclipse (ADT)

In this module, we'll talk about obtaining and installing these tools, and ensuring that your development environment is ready for the course assignments.

Objectives

By the end of this module, you will be able to:

  • Identify the tools needed for Android application development
  • Locate and install your development tools
  • Create a simple 'Hello, World' application
  • Execute the application on an Android emulator

Learning Guide

Begin this module by watching all of the recorded lectures below. I strongly recommend that you watch them in the order listed, as some lectures assume knowledge gained from earlier lectures.

Refer to the Course Outline for any assignment start or due dates!
