Module 17 - Files

605.486 - Mobile Application Development for the Android Platform

Overview

We've seen how data can be persisted in a relational database. Sometimes data is more linear, such as audio files or a text document. File processing in Android is very similar to standard Java File I/O.

In this module, we'll read and write files, keeping them private to an application, or making them available to other applications. We'll also use external storage, such as SD cards to allow easy transfer of files between devices and computers.

Objectives

By the end of this module, you will be able to:

  • Read and write files that only a specific application can access.
  • Read and write files that other applications can access.
  • Read and write files using an SD card as storage.

Learning Guide
