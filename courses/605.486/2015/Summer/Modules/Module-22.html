Module 22 - RESTful Web Services

605.486 - Mobile Application Development for the Android Platform

Overview

Sometimes your applications need to access data or services that are not located on the device itself. Banking applications, for example, need to interact with account information on a server.

In this module, we'll discuss how to communicate with a server using RESTful web services, passing data between the server and your application using JavaScript Object Notation (JSON).

Objectives

By the end of this module, you will be able to:

  • Describe what a RESTful web service is.
  • Write code to call RESTful web services from your Android applications.
  • Parse and create JSON data streams to interact with a RESTful web service.
