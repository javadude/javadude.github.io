Module 21 - Navigation

Overview

An application is rarely made up of only a few Activities. Providing effective navigation for your users is key to a good experience.

In this module, we'll use the ActionBar's navigation capabilities to display and switch between multiple Fragments in an Activity. Finally, we'll create a Navigation Drawer that allows easy access to key locations in your application.

Objectives

By the end of this module, you will be able to:

  • Add tabbed, swiping and drop-down navigation to an ActionBar.
  • Create a Navigation Drawer that slides out when needed.

Learning Guide

Begin this module by watching all of the recorded lectures below. I strongly recommend that you watch them in the order listed, as some lectures assume knowledge gained from earlier lectures.
