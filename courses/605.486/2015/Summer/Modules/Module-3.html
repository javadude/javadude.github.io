Module 3 - Intents

605.486 - Mobile Application Development for the Android Platform

Overview

Most applications require more than one Activity. In Android, you start activities via “Intents”, describing either the target Activity or simply the action you want to perform.

In this module, we'll explore how you can use explicit and implicit Intents to launch applications, and return results to the caller.

Objectives

By the end of this module, you will be able to:

  • Describe the difference between explicit and implicit Intents.
  • Use explicit and implicit Intents to start Activities.
  • Create Intent filters in your application manifest to catch implicit Intents.
  • Describe the stack mechanism used to keep track of navigation between Activities.
