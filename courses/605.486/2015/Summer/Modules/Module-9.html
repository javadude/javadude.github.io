Module 9 - Date and Time Pickers

Overview

Some user inputs should be constrained, and alternate user-interface components can make their entry much simpler. Inputting a date or time can be error prone and tedious. The Android libraries provide Date and Time picker components to simplify entry.

In this module, we'll explore entering dates and times using “pickers”, displayed in pop-up dialogs.

Objectives

By the end of this module, you will be able to:

  • Display existing date and time values via formatted text.
  • Create dialogs containing date and time pickers to allow modification of their values.
  • Retrieve the values safely in an activity after from the dialogs.
