Module 11 - Manipulating Graphics

605.486 - Mobile Application Development for the Android Platform

Overview

Drawing graphics is a start, but often the user should be able to interact with those graphics.

In this module, we'll build a simple graph editor, allowing a user to create, move and connect shapes.

Objectives

By the end of this module, you will be able to:

  • Create custom shape drawables.
  • Define shapes in XML and attach them to ActionBar buttons.
  • Respond to user touch.
  • Dynamically add graphical objects to the screen.
  • Respond to user drag gestures to move shapes.

Learning Guide

Begin this module by watching all of the recorded lectures below. I strongly recommend that you watch them in the order listed, as some lectures assume knowledge gained from earlier lectures.
