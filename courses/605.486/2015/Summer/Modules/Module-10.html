Module 10 - Graphics

Overview

Some applications, such as contact managers and to-do lists may be effectively represented by simple user-interface components. Other applications, especially games, require more control over what is displayed to the user. Android supports multiple APIs for graphical manipulation, giving you complete control of the screen.

In this module, we'll use the basic Android Canvas support to draw on the screen and perform simple animation.

Objectives

By the end of this module, you will be able to:

  • Create a custom View class to display graphics in a UI.
  • Draw shapes on the screen using simple graphics primitives in that View class.
  • Animate shapes on the screen using a separate thread to update their position.
  • Properly size the custom View based on layout parameters such as WRAP_CONTENT and MATCH_PARENT.
  • Rotate, scale and fade the custom view via code or XML.
