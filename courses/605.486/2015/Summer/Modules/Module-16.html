Module 16 - Services

605.486 - Mobile Application Development for the Android Platform

Overview

Sometimes you need background processing that isn't tied to a visible application, or provide a common front-end for a resource or common function.

In this module, we'll explore the Services component, connecting Activities to it to start/stop background processing, and binding to it to access resources and perform utility functions.

Objectives

By the end of this module, you will be able to:

  • Create a 'started' service.
  • Create a 'bound' service.
  • Start/stop a service.
  • Bind to a local service (same application on device).
  • Bind to a remote service (separate application on device).
  • Use Handlers or AIDL for remote service communication.
