VisualAge Friendly Beans

If you want your bean added to this list, send me an email (scott@javadude.com) and I'll add it!

The following beans are a few that work nicely in VisualAge for Java:

  • TabSplitter - a nifty tabbed panel component
  • Box Beans - a set of beans to assist use of BoxLayout
  • BorderEditor - a JavaBean property editor that makes Swing Borders easy to use in VisualAge for Java

More will be added as folks send them to me!
