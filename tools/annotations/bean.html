JavaDude Annotations: @Bean

[Back to main JavaDude Annotations page]

You define a JavaBean using the @Bean annotation. This annotation works together with an annotation processor to generate a superclass. For example,

package com.javadude.annotation.example;


import com.javadude.annotation.Bean;


@Bean(...)  


public class Person extends PersonGen {

}

First, you add an @Bean annotation to your class. This will trigger the annotation processor to generate a superclass. To take advantage of this superclass you must add extends ClassNameGen to your class declaration. The name of the generated superclass will be the same as the annotated class with "Gen" added to it, and it will be generated in the same package as the annotated class.
