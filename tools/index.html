Tools

  • JavaBean Annotations
    Generate getters/setters, delegate methods, equals/hashCode, mixins, default method parameters
     
  • Dynamic Working Sets for Eclipse
    Working sets that automatically find their content based on patterns or project natures
     
  • ANTXR (XML Parser)
    XML parsing based on an ANTLR-like grammar
     
  • Eclipse-Jindent
    Jindent integration for eclipse
     
  • SWT Layouts
    Some common AWT layouts converted for use in SWT
     
  • Border Editor
    A JavaBean property editor for Swing borders
     
  • ParseView
    A visual debugger for ANTLR 2.x
     
  • Box Beans
    JavaBean wrappers for Swing's BoxLayout making it easy to use in bean builders
     
  • TabSplitter
    A really old JavaBean that combines a tabbed panel and split pane function
     
  • SplitterLayout
    A layout manager that creates split-pane functionality
     
  • AutoGut
    Extracts an interface from a class in VisualAge for Java (before refactoring was cool!)
     
  • Importifier
    Expands import-on-demand (.*) statements in VisualAge for Java (like Organize Imports in eclipse)
